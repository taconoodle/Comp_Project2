VM είναι το Virtual Machine. Δημιουργείς ουσιαστικά ένα εικονικό μηχάνημα και προγράμματα και μετά τρέχεις τα προγράμματα.
Το πρόγραμμα διαλέγει αυτόματα ποιο VM είναι το καλύτερο VM για να τρέξει το πρόγραμμα.
Μέσα στον φάκελο cfg έχει δύο αρχεία config. Κάνε δεξί-κλικ -> άνοιγμα με notepad και άλλαξε τις τιμές στη μορφή που είναι.
Κανονικά δεν πρέπει να το ενοχλούν τα spaces ανάμεσα στις λέξεις, τέσταρέ το και πες μου αν βγάζει θέμα με αυτό.
Για να εισάγεις προγραμματα και VM από τα αρχεία αυτά πάτα τα import κουμπιά.
Τα προγράμματα έχουν απαιτήσεις πυρήνων επεξεργαστή, RAM, SSD χώρο, network bandwidth και αριθμό GPUs. Τα VM έχουν τους αντίστοιχους πόρους, συν OS.
Πρόσεξε ότι το ελάχιστο Bandwidth που θα αναθέσεις σε ένα VM πρέπει να είναι 4 GB/s.
Τα προγράμματα δεν τρέχουν πραγματικά. Απλώς όταν τα δημιουργείς τους βάζεις έναν εκτιμώμενο χρόνο εκτέλεσης και όταν πατάς execute απλά μετράει ένα timer για το καθένα.
